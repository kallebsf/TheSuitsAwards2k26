# SUITS Awards

O **SUITS Awards** é uma comemoração criada entre amigos para celebrar histórias, momentos marcantes e pessoas que fizeram diferença ao longo dos anos. Não é um prêmio corporativo, não é algo formal — é memória, zoeira, reconhecimento e história compartilhada.

O site existe para registrar tudo isso em um só lugar.

## O que é o SUITS Awards

Tudo começou como uma ideia simples: reconhecer, de forma simbólica, quem marcou época no grupo. Com o tempo, virou tradição. Cada edição representa um ano vivido, com seus acontecimentos, conquistas, erros, piadas internas e personagens que ficaram na história.

É mais sobre lembrar do que ganhar.

## O que o site faz

- Apresenta a história do SUITS Awards  
- Lista todas as categorias criadas pelo grupo  
- Destaca a principal premiação: **Homem do Ano**  
- Centraliza votações e resultados  
- Funciona como arquivo histórico do grupo

## Importante Deixar Claro

Este projeto não tem fins comerciais.  
Não é uma competição séria.  
Não é um ranking oficial.

É uma comemoração entre amigos, feita para registrar momentos, manter tradições e dar risada no futuro lembrando de tudo isso.

## Futuro do Projeto

Enquanto existir amizade, o SUITS Awards continua evoluindo. Novas categorias, novos anos, novas histórias e, inevitavelmente, novas polêmicas internas.
